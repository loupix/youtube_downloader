# Youtube Downloader

Plateforme web pour télécharger les vidéos de Youtube, Dailymotion, etc ..

## Installation

Créer l'environnement virtuel et installer les composants

- `python -m venv env`
- `./env/bin/activate`
- `pip -r requirements.txt`

Installer les composants AngularJS

- `bower install`

Et Enjoy
